# Your first program - finished project

The completed code from the [Your first program](https://scratch4j.openpatch.org/tutorials/getting-started)
tutorial.

## Running it

1. Download the Scratch for Java jar from https://scratch4j.openpatch.org/download
2. Put it in the `+libs` folder next to this file
3. Open this folder with BlueJ, right-click `MyStage` and choose `new MyStage()`

## Assets

None. Every costume, backdrop and sound this project uses is built into
Scratch for Java and is referred to by name, so there are no image or
sound files to keep track of.
