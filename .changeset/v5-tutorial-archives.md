Add a finished project for every tutorial.

Each of the five tutorials now links a downloadable project containing the
completed code, in the same way `bouncy-hedgehog-100` already did:

::archive[Project: Make it Walk 100%]{name="make-it-walk-100"}

They are useful in two directions — a learner whose code will not run can
compare against a working version, and a teacher can check a chapter end to end
without typing it out.

Because these tutorials use only built-in costumes and sounds, the archives hold
nothing but the `.java` files: between 1.7 and 2.3 kB each, against 104 kB for
the asset-carrying hedgehog project.
